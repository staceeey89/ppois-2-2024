# Документация программной системы

## Введение

Эта программная система представляет собой систему, имитирующую модель повторной сдачи экзамена.
включающую в себя различные классы для представления студентов, преподавателей, экзаменов и т.д.

## Классы и методы

### Класс CLI

Класс является основным интерфейсом пользователя для взаимодействия с системой. Он предоставляет пользователю меню с выбором действий и обрабатывает пользовательский ввод.

#### Основные методы

-`__init__(self)`: Инициализирует экземпляр класса CLI, создавая экземпляр класса ExamSystem.
- `run(self)`: Запускает основной цикл программы, который отображает меню доступных действий и обрабатывает ввод пользователя.
- `take_test(self)`: Позволяет студенту пройти тест, вводя имя и фамилию студента, а также название экзамена.
- `consult_with_teacher(self)`: Позволяет студенту консультироваться с преподавателем, вводя имя и фамилию студента, а также предмет консультации.
- `add_score(self)`: Добавляет оценку студенту, вводя имя и фамилию студента, а также название экзамена и оценку.
- `check_passed(self)`: Метод для проверки сдачи экзамена. Он запрашивает имя и фамилию студента, название экзамена и выводит результат сдачи.
- `check_expulsion(self)`: Метод для проверки отчисления студента. Он запрашивает имя и фамилию студента и выводит результат отчисления.
- `add_learning_material(self)`: Метод для добавления учебного материала студенту. Он запрашивает имя и фамилию студента, название предмета и название материала, а затем добавляет его к студенту.
- `add_additional_classt(self)`: Метод для добавления дополнительного занятия студенту. Он запрашивает имя и фамилию студента, название занятия и дату, а затем добавляет его к студенту.
- `analyze_student(self)`: Метод для анализа информации о студенте. Он запрашивает имя и фамилию студента и выводит полный отчет о студенте.
- `delete_student(self)`: Метод для удаления студента из системы. Он запрашивает имя и фамилию студента и удаляет его из системы.
- `is_valid_name(self)`: Статический метод для проверки корректности имени или фамилии. Он использует регулярное выражение для проверки, что имя или фамилия состоят только из букв.

### Класс Exam

Класс для представления экзамена в системе управления экзаменами. Он хранит информацию о названии экзамена, проходном балле и списках оценок, полученных студентами.

#### Основные методы

- `__init__(self, name: str, passing_score: int)`: Инициализирует экземпляр класса Exam, создавая объект с именем экзамена и проходным баллом.
- `add_score(self, score: int)`: для добавления оценки к экзамену. Он добавляет оценку в список оценок экзамена.
- ` last_score(self)`: для получения последней оценки, полученной на экзамене
- `has_passed(self)`: Метод для проверки, сдал ли студент экзамен. Он проверяет, является ли последняя оценка не менее проходного балла.
- `__str__(self)`: Метод для представления экзамена в виде строки. Если у экзамена нет оценок, он выводит сообщение "нет оценок", иначе выводит список оценок.

### Класс ExamSystem

Класс представляет собой систему управления экзаменами. В нем хранятся списки студентов и преподавателей, а также словарь SUBJECTS, который содержит преподавателей по предметам.

#### Методы
- `__init__(self)`: Инициализирует экземпляр класса ExamSystem и создает пустые списки для студентов и преподавателей.
- `add_student(self, student: Student)`: Метод для добавления студента в систему.
- `get_student(self, first_name: str, last_name: str)`: Метод для получения студента по его имени и фамилии. Возвращает объект студента, если он найден, и None, если нет.
- `delete_student(self, first_name: str, last_name: str)`: Метод для удаления студента из системы по его имени и фамилии.
- `get_teacher(subject: str):`: Статический метод для получения преподавателя по предмету. Возвращает объект преподавателя, если он найден, и None, если нет.
### Класс LearningMaterial

Класс для представления учебного материала для подготовки к экзаменам

#### Методы

- `__init__(self, title: str, material_subject: str)`: Инициализирует экземпляр класса LearningMaterial с заголовком и предметом учебного материала.
- `use_for_preparation(self)`: Метод для использования учебного материала для подготовки к экзамену. Выводит сообщение о том, какой учебный материал используется для подготовки к экзамену.

### Класс Student

Класс для представления студента в системе управления экзаменами.

#### Методы

- `__init__(self, first_name: str, last_name: str, group: str)`:  Инициализирует экземпляр класса Student с заданными именем, фамилией, номером группы и пустыми списками и словарем для хранения экзаменов, учебных материалов, дополнительных занятий и консультаций.
- ` add_exam(self, exam: Exam)`:Добавляет экзамен в словарь экзаменов студента.
- `validate_group_number(group: str)`:  проверяет, является ли номер группы студента валидным.
- `add_score(self, exam_name: str, score: int)`: Добавляет оценку к экзамену студента.
- `add_learning_material(self, material: LearningMaterial)`:Добавляет учебный материал в список учебных материалов студента.
- `attend_additional_class(self, additional_class: AdditionalClasses)`Добавляет дополнительное занятие в список дополнительных занятий студента.
- `add_consultation(self, teacher_name: str, subject: str)`:Добавляет консультацию в список консультаций студента.
- `analyze(self)`: Возвращает полный отчет о студенте, включая информацию об экзаменах, учебных материалах, дополнительных занятиях и консультациях.
- `analyze_test(self, exam_name)`: Возвращает отчет о тесте студента по заданному предмету, включая информацию об учебных материалах и дополнительных занятиях по этому предмету.
- `def __str__(self)`: Возвращает строковое представление студента.

### Класс Teacher

Класс для представления преподавателя в системе управления экзаменами

#### Методы

- `def __init__(self, first_name: str, last_name: str)`: Инициализирует экземпляр класса Teacher с заданным именем и фамилией.
- `def consult_with_student(self, student_name: str)`:  Выводит сообщение о том, что преподаватель консультирует заданного студента.

### Класс Additional_classes

Класс для представления дополнительного занятия в системе управления экзаменами. 

#### Методы

- ` __init__(self, title: str, date: str)`: Инициализирует экземпляр класса AdditionalClasses с заданным названием и датой. Метод проверяет валидность даты с помощью статического метода is_valid_date
- `conduct_class(self)`: Выводит сообщение о том, что дополнительное занятие проводится в заданную дату.
- `is_valid_date(date: str)`:Статический метод, проверяющий валидность даты в формате dd.mm.yyyy.
